# Payment Admin Dashboard Backend

Backend API for the payment admin dashboard.

## Getting Started

1. Create a `.env` file using `.env.example`.
2. Install dependencies: `npm install`.
3. Run the database: `docker compose up -d postgres`.
4. Run migrations: `npm run prisma:migrate`.
5. Start the server: `npm run dev`.

## Scripts

- `npm run dev` - start in watch mode
- `npm run build` - build for production
- `npm run start` - run the built app
- `npm run lint` - lint
- `npm run format` - format

## API Docs

Swagger UI is available at `/docs` when the server is running.
